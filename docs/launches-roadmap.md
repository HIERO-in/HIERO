# Launches 미래 기능 ROADMAP

> 지금은 만들지 않는다. 하지만 잊지 말 것.  
> heiro가 단계별 콘텐츠 검토 시 메모한 미래 기능들.

---

## Phase 2: 위치 기반 자동화

### 2.1 자동 지도 + 주변 분석 (물건탐색·현장확인)
- 주소 입력 → Naver Maps 또는 Kakao Maps API 자동 표시
- 도보 5분/10분 반경 자동 분석:
  - 지하철역 거리
  - 편의점/마트
  - 병원/약국
  - 학교
  - 카페/음식점
  - 공원
- 주변 호텔/숙박 가격 자동 크롤링 → 경쟁 가격대 파악
- 생활권 평균 대비 위치 점수 자동 산출

### 2.2 풀 매칭 정밀화
- properties 테이블에 `district`, `grade` 컬럼 추가
- reservations + transactions 조인으로 풀 평균 ADR/점유율 정확 계산
- 신규 물건이 풀에 들어가면 예상 매출 +X% 자동 계산
- 같은 풀의 호실 시각화 (지도 위 핀 + 거리 평균)

## Phase 3: 계약 자동화

### 3.1 등기부등본 자동 발급
- 국토교통부 일사편리 API 또는 공인된 등기부 발급 서비스 연동
- 주소 입력 → 등기부 자동 발급 (PDF 다운로드)
- 자동 분석:
  - 임대인 명의 추출
  - 선순위 권리 (근저당, 가압류) 검출
  - 위험 신호 ⚠ 자동 표시

### 3.2 계약서 사진 OCR
- 계약서 사진 업로드 → Naver Clova OCR 또는 Google Vision
- 자동 추출:
  - 보증금 / 월세 / 관리비
  - 임대 기간
  - 특약 사항
  - 임대인 / 임차인 정보
- 추출된 데이터로 계약진행 단계 필드 자동 채움

### 3.3 전자계약서 자동 생성 + 승인
- 모두싸인 / 이폼사인 / docusign API 연동
- HIERO에서 계약 조건 입력 → 전자계약서 자동 생성
- 임대인 이메일로 자동 전송 → 모바일 사인
- 승인되면 PDF 자동 보관 + 잔금 단계로 자동 advance

## Phase 4: 운영 자동화

### 4.1 청소 자동 배정 (띵동 통합)
- 체크아웃 감지 → 청소 자동 배정
- 띵동 API (있는 경우) 또는 카톡/이메일 템플릿 자동 생성
- 청소 완료 사진 자동 수신 + 비용 transactions 자동 등록

### 4.2 가격 동적 조정
- 풀 ADR 기반 자동 가격 제안
- 시즌/이벤트/요일별 동적 가격
- PriceLabs 통합 검토

### 4.3 자동 메시지
- 체크인 D-1: 입실 안내 자동 발송
- 체크아웃 D+0: 리뷰 요청 자동 발송
- 게스트 언어 자동 감지 (한/영/중/일)

---

## Phase 5: 분석 + 인텔리전스

### 5.1 생활권 수요 패턴 분석
- 시즌별 점유율 변화
- 지역별 이벤트 (행사, 콘서트, 컨퍼런스) 영향 분석
- 신규 호실 매물 추천 (높은 수요 + 낮은 공급 조합)

### 5.2 게스트 LTV (Lifetime Value)
- 재방문 게스트 추적
- 게스트별 누적 매출
- VIP 게스트 자동 태깅

### 5.3 호실별 P&L 자동 진단
- 손실 호실 자동 알림
- 가격 / 청소비 / 셋팅 비용 최적화 제안
- 풀 재배치 권고 (같은 풀에서 재고 조정)

---

## Phase 6: 멀티 운영자

### 6.1 권한 시스템
- 운영자별 역할 (heiro / 청소담당 / 회계 / 게스트응대)
- 단계별 담당자 자동 알림

### 6.2 협업 워크플로우
- 단계별 검토 + 승인 흐름
- 운영 일지 자동 생성

## 우선순위 결정 기준 (참고)

1. **운영 시간 절약** — heiro의 일일 시간을 가장 많이 쓰는 작업부터
2. **데이터 신뢰성** — 수동 입력 → 자동 데이터로 전환 (오류 감소)
3. **확장성** — 호실 100개 → 1,000개로 늘어났을 때 필수가 되는 기능
4. **수익 영향** — 가격 최적화, 공실 감소처럼 직접 매출 영향

각 Phase 진입 결정 시 위 4가지 기준으로 우선순위 평가.
